Tiempos de espera

Posiblemente te ha pasado que estas en un NavBar navegando entre las diferentes categorías y cuando pasas de una a otra se pueden cerrar de manera muy rápida ya que el cursos no esta totalmente alineado con el área del efecto.

Con este tip se busca darle un poco más de tiempo al usuario para que sus movimiento no estén muy condicionados a la precisión del cursor.
